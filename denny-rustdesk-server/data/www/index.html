Quick Guide to Setting Up Connections Between Client and Server

 

  1. Open RustDesk on the Client
    Launch RustDesk on the computer or device you want to use.

  2. Open Settings
    Click on the three-dot menu next to your ID and select Settings.

  3. Unlock Network Settings
    In the settings menu, go to the Network section. Click on Unlock Network Settings. You may need to enter your system password to modify these settings.

  4. Enter Server Information

    • ID Server: Enter the IP address or Domain of your RustDesk Relay server.

    • Key: Enter the Public Key you obtained during the server installation (this was shown to you after the server installation).

  5. Apply the Settings
    After entering the details, click Apply to save the settings.

  6. Repeat for All Clients
    Perform the same steps on all devices that should connect to your server. You need to enter the same IP/Domain and Public Key on all of them.

 

For more detailed information about RustDesk, including server setup, user guides, and developer documentation, you can visit the RustDesk Documentation.
