Quick Guide to Setting Up Connections Between Client and Server 🔧 

 

  1. Open RustDesk on the Client
    Launch RustDesk on the computer or device you want to use.
  2. Open Settings
    Click on the three-dot menu next to your ID and select Settings.
  3. Unlock Network Settings
    In the settings menu, go to the Network section. Click on Unlock Network Settings. You may need to enter your system password to modify these settings.
  4. Select "ID/Relay Server"
    After unlocking the network settings, select the "ID/Relay Server" option.
  5. Enter Server Information
    • ID Server: Enter umbrel.local:21115 as the IP address or Domain of your RustDesk Relay server.
    • Key: Enter the Public Key you obtained during the server installation. To get the Public Key, right-click on the RustDesk app in Umbrel, select the "Troubleshoot" option, and find the key in the logs.
    • API Server: This field can be left empty.
    • Relay Server: This field can be left empty.
  6. Apply the Settings
    After entering the details, click Apply to save the settings.
  7. Repeat for All Clients
    Perform the same steps on all devices that should connect to your server. You need to enter the same umbrel.local:21115 and Public Key on all of them.

💡 For more detailed information about RustDesk, including server setup, user guides, and developer documentation, you can visit the RustDesk Documentation.
